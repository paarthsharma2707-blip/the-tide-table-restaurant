# the-tide-table-restaurant
A responsive restaurant landing page built with HTML, CSS, and JavaScript featuring an elegant seafood restaurant design, interactive reservation system, dynamic menu, and modern responsive UI.
## ✨ Features

- 🍽️ Elegant restaurant landing page
- 📖 Interactive menu section
- 📅 Table reservation system
- 👥 Party size selector
- 🕒 Time-slot picker
- 🎉 Upcoming events section
- 📱 Fully responsive layout
- 🎨 Modern and premium UI design
- ⚡ Smooth scrolling navigation

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- SVG Graphics

This project demonstrates:

- CSS Grid & Flexbox
- JavaScript DOM Manipulation
